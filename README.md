# Titanic Survival Prediction — Feature Engineering Pipeline

## Objective
Build a reproducible machine learning workflow demonstrating feature engineering and pipeline design.

## Workflow
- Data Cleaning
- Feature Engineering
- Baseline Model
- Enhanced Model
- Model Evaluation
- Ethical and Leakage Analysis

## Engineered Features
- Age binning
- Log fare transformation
- Interaction feature (Pclass × Fare)
- Encoding and scaling via pipeline

## Results
The engineered model achieved higher accuracy than the baseline model, demonstrating the impact of feature engineering.

## Reproducibility
All preprocessing steps are implemented using sklearn Pipelines.
